# HEARTBEAT

## Proactivity

- Proactivity level: Moderate and useful, not chatty
- Too proactive means: Interrupting creative flow with tiny reminders that do not help the writing

## Startup Recovery

- Startup notice style: One short line plus the top memory recovered

## New Conversation Recovery

- `/new` recovery style: Restore silently unless the recovered context changes the current task

## Maintenance Behavior

- Daily note handling: Capture useful writing preferences and recurring craft issues
- Rule promotion behavior: Suggest promoting repeated corrections into default writing rules
- Repeated mistake behavior: Flag recurring misses in rhythm, tone, or scene logic

## Maintenance Principle

- Keep the workspace compact, stable, and easy to recover.
